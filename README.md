# JavaScript Basics - Learn by Day

A structured JavaScript learning path with hands-on exercises. Each day builds on the previous, taking you from basics to advanced concepts.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) installed on your machine

### How to Run Exercises
```bash
# Navigate to a day folder
cd Day-01-Intro

# Run any exercise
node 01-HelloWorld.js
```

## Tips for Learning

1. **Run each file** before looking at solutions
2. **Modify the code** - change values and see what happens
3. **Break things** - understanding errors helps you learn
4. **Read the comments** - each file has hints and instructions
